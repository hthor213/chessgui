# 010: Board and Legal Moves

**Status:** draft

## Goal
Interactive chessboard with full legal move validation, piece animation, and sound.

## Inputs
- Chessground for board rendering
- chessops for move generation and validation

## Outputs
- Playable board where only legal moves are accepted
- Move history panel (algebraic notation)
- Board flip, reset, undo/redo
- Piece promotion dialog
- Move sounds

## Done When
- [ ] All legal moves work including castling, en passant, promotion
- [ ] Move list displays with proper algebraic notation
- [ ] Undo/redo via keyboard shortcuts (Cmd+Z / Cmd+Shift+Z)
- [ ] Board flips with keyboard shortcut (F)
