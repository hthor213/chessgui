# 102: Standard Keyboard Shortcuts

**Status:** draft

## Goal
Add standard keyboard shortcuts for common actions, matching user expectations from Lichess and desktop apps.

## Approach
- Extend the existing keydown listener in App.tsx
- Add shortcut hints to UI elements (tooltips)
